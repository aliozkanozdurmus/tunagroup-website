# Tuna Group Website

Client-facing website repository for Tuna Group.

## Overview

This repository is prepared for client delivery with:

- clean and minimal commit history
- baseline documentation and contribution standards
- secret-safe defaults for environment configuration

## Quick Start

1. Install dependencies for the project stack.
2. Copy .env.example to .env.local.
3. Start the local development server with project scripts.

## Security

- Do not commit API keys, tokens, private keys, or .env files.
- Keep sensitive values only in local or platform secret stores.
